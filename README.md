# Movie Review App

## Overview
This is an Android app built using Kotlin, designed to manage a collection of movies. The app allows users to view a list of movies, add new movie reviews, and display average ratings for the movies. The main features of the app include a splash screen, a main screen with buttons to add and view movies, and a detailed screen to view movie information.

## Features
- **Splash Screen**: Displays the app's logo while it loads and transitions to the main screen after a few seconds.
- **Main Screen**: 
  - Displays buttons for adding a new movie and viewing movie reviews.
  - Shows the average rating for movies currently in the collection.
- **Detailed View Screen**: 
  - Displays detailed information about each movie (title, director, rating, and comments).
  - Allows users to navigate back to the main screen.
  
## Technologies Used
- **Kotlin**: Programming language used for app development.
- **Android Studio**: IDE used for app development and testing.
- **Arrays**: Used for storing movie data (title, director, rating, comments).
